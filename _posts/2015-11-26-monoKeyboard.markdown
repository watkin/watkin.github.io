<h4>查看详细快捷键： </h4>
菜单--MonoDevelop-Unity  ->Preference ->Key Bindings   
<h4>常用</h4>
Command+' ---------unity API Reference<br>
Command+/ ---------切换行注释<br>
Alt+up/down ---------切换行<br>
Alt_Left/Right --------定位下一个单词<br><br>
Command+F   --------查找当前脚本<br>
Command+Shift+F ------查找全部脚本<br><br>
Command+y  -------进入到焦点对应的函数<br>
Command+w  -------关闭类<br>
Command+s  -------保存<br>
